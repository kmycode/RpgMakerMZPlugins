# RpgMakerMZPlugins

雪あすか（ＫＭＹ）の制作したRPGツクールMZ向けのこまこまとしたプラグインを置いときます。

役に立つか分かりませんが、全てライセンス不記載で自由に使って構いません。

このようなこまこましたものは誰が作っても同じようなものになりますから、自作発言についても特に禁止はしませんが、バグとかあっても私のせいにしたり、私の意見を求めることを修正における必須プロセスにしたり、著作権表記が必要なライセンスをつけて私の作るゲームに表記を要求したりその他金銭など要求したりしなければ何でもいいです。

ただバージョン番号は多分更新さぼるので、プラグインをダウンロードした日付を適宜記録しておいて、アップデートが必要か各自判断をお願いします。
